ProductiveTalkのセッション

キャリアコンサルタントとのProductive Talk を通じて、ご自身のキャリアの実感を深め、ご自分らしい未来のキャリアを創りだしていく。そのための具体的なセッションをご案内します。

自己理解を深め、キャリア満足度を高める

☆Mercury Session

　　～キャリアを振り返るセッション～

　これまで歩いてきた道筋や転機を振り返り、そこにある自分だけのキャリアストーリーと感じるセッションです。
あなたが語る言葉のから、あなたのキャリアストーリーが立ち上がってきます。

☆Venus Session

　　～価値観を見える化する～

　自分の人生で大切にしている価値観を、セッションを通じて、言葉にして実感していきます。
価値観が見えてくると、今までの転機はもちろん、今、これからの人生の機会や選択がよりご自分らしいものになっていきます。

☆Mars Session

　　～ポジティブコアをつかむ～

　過去の成功体験を振り返る中から、自分のポジティブコア（ゆるがない強み）をつかんでいきます。
強みを自分が知り、自覚することで、よりこれからの人生で強みを発揮、発展させることができます。

未来のキャリア幸福度を高める

☆Jupitar Session

　　～これからのキャリア展望を持つ～

　どんなことを実現していきたいか、どんな方向に進んでいきたいか、対話する中からありたい未来のご自身の姿をゴールセッティングし、実現へのストーリーを描きます。

☆Uranus Session

　　～しなやかに学ぶ力を高める～

　自分の学びスタイルを知ること、そして、新しい学びのスタイルを探索することで、しなやかに学び未来の自分を変える力を高めていきます。
